# 更新记录（Changelog）

作者：CC

## v1.3.0（2026-02-26）
- 检测：服务级自动检测开关持久化（纳管后默认关闭；开启后才创建定时检测任务）
- 策略：失败策略开关（失败告警/自动重启）持久化与弹窗可视化配置
- 增强：自动重启后延迟复检（用于“重启后需要启动时间”的服务）
- 修复：自动检测弹窗“保存卡顿/保存不上”与刷新并发问题；手工检测不触发自动重启；自动重启成功不再写入错误日志

## v1.2.0（2026-02-26）
- UI：服务表格横向滚动修复与筛选增强（关键字/类别/失败策略/状态），项目信息弹窗（版本/更新时间/更新记录）
- 运维：复杂启停命令支持 `@script:` 本地脚本自动上传远端执行
- 稳定性：启动阶段全量检查改为后台执行、引擎层异常兜底、存储原子写

## v1.1.0（2026-02-25）
- 增加服务级“只监控/可运维”防误操作开关
- 补全配置参考与用户手册

## v1.0.0（2026-02-24）
- 初版：支持 HTTP 健康检查、SSH 启停与 Web 运维界面
